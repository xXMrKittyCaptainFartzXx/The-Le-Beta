PROJECT SERVAL - Character Instrument

The fuck is this?

Project SERVAL (Synthetic Expressive Reactive Visual-Audio Littleguy) is a custom "Visual Character Instrument" project yours truly is creating using DIY Analog & DSP modular synthesis to drive character animations via OSC into Godot.

Think of it as music signals & data operating a furry puppet!

Essentially the idea is to use control voltages and digital data to make a character instrument come to life. The project's acronym comes from literally using pixel art I made of a serval character (that I created in Krita & Pixelorama) and making it do animations when sound go BRRRRR.

If you want to watch this thing in action, I'm currently working on a MVP (minimal viable product) version of this with just DSP (analog to come) starting with a custom VCO ("voltage" controlled oscillator, although in this case I guess it's digital-driven? lmao).

You can just watch the video when it comes out (although it's gonna be obnoxious and pitch-driven w/o amplitude variation since it's just a "VCO"). If you want to know more about the technical details, I'll leave descriptions of it linked to what I'm doing in each iteration of Project SERVAL.

For now uh, sit back and be patient, motherfucker. I'm currently working through some avalanche/bootstrap/voltage comparator designs to start it up with a sawtooth in falstad to create custom Arrays. If you don't understand what that means - I basically need make sound vibrate in particular ways then push it into software for custom stuff. Then i can make this thing actually start to move.
